# itz
Задание: 
Необходимо загрузить три исходных файла:
1. Сводный файл прошлого года. Это основной источник для копирования распределения преподавателей. В нем нагрузка уже сведена, а ФИО преподавателей уже расставлены.
2. Файл текущего года «Осень». Это файл с текущей нагрузкой на осенний семестр. В начале года он пустой по преподавателям, то есть ФИО в нем отсутствуют или заполнены частично.
3. Файл текущего года «Весна». Это файл с текущей нагрузкой на весенний семестр. В начале года он также пустой по преподавателям, то есть ФИО в нем отсутствуют или заполнены частично.

Работа моего кода:
Мой код написан в виде класса Loader.В данном случае класс отвечает за загрузку трёх Excel-файлов и показ информации о них.
Что происходит при запуске программы:
Сначала я подключаю библиотеку pandas. Эта библиотека нужна, потому что обычный Python не умеет читать Excel-файлы. Pandas добавляет такую возможность.
Дальше я описываю класс Loader. Внутри класса есть несколько методов.
Метод init — В нём я сохраняю пути к трём файлам и номера листов, которые нужно загрузить. Также я создаю три пустых места None для будущих таблиц.
Метод load_excel — загружает один Excel-файл. Он принимает имя файла и номер листа. Внутри используется блок try-except для обработки ошибок. Если файл найден и лист существует, pandas читает файл и возвращает таблицу. Если файл не найден — выводится сообщение об этом. Если нет листа с таким номером — тоже сообщение. Если любая другая ошибка — выводится её текст. В случае ошибки метод возвращает None.
Метод print_info — выводит информацию о таблице на экран. Если таблица пустая, он просто пишет «данные не загружены» и выходит. Если таблица есть, он показывает: размер таблицы (строки и столбцы), названия всех столбцов, первые 5 строк и типы данных в каждом столбце.
Метод load_all — загружает все три файла по очереди. Сначала загружается файл прошлого года и выводится его информация. Потом осенний файл и его информация. Потом весенний файл и его информация. В конце печатается сообщение «Загрузка завершена».

После описания класса я создаю объект loader и передаю ему имена трёх файлов. Файлы у меня имеют расширение .xls . Затем я вызываю метод load_all, который и запускает всю загрузку.
Из библиотеки pandas в моём коде используется только несколько вещей:
pd.read_excel — функция, которая читает Excel-файл и превращает его в таблицу
.shape — свойство таблицы, которое показывает её размер (сколько строк и столбцов)
.columns — свойство, которое показывает названия всех столбцов
.head() — метод, который возвращает первые 5 строк таблицы
.dtypes — свойство, которое показывает типы данных в каждом столбце (числа, текст, даты)

Что получилось в итоге
В итоге программа создаёт объект-загрузчик, передаёт ему три файла, загружает каждый из них и показывает на экране: размер таблицы, имена столбцов, первые 5 строк и типы данных. Если какого-то файла нет или лист не найден, программа не падает, а сообщает об ошибке понятными словами.
